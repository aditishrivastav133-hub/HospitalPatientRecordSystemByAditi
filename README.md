# Hospital Management System

A Java-based Hospital Management System that helps manage hospital operations such as patient records, doctor details, appointments, and other healthcare-related information. The project uses MySQL database integration for storing and managing data.

## Technologies Used

- Java
- MySQL
- JDBC (Java Database Connectivity)
- SQL

## Features

- Patient Management
  - Add new patients
  - View patient details
  - Update patient information
  - Delete patient records

- Doctor Management
  - Add doctor details
  - View doctor information
  - Manage doctor records

- Appointment Management
  - Schedule appointments
  - Store appointment details

- Database Integration
  - MySQL database used for storing hospital data
  - SQL file included for database creation
